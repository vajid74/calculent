Calculent Logic provides a calculation engine with REST endpoints, shortcodes, and admin settings for WordPress.

== Description ==
- Calculator class for evaluating expressions.
- Shortcode to embed calculators.
- REST API endpoint with nonce protection.
- Admin settings for feature toggles.

== Installation ==
1. Upload the `calculent-logic` folder to `/wp-content/plugins/`.
2. Activate the plugin through the "Plugins" menu in WordPress.
